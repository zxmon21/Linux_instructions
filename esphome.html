ESPHome + Docker Setup

  1. Install prerequisites first 
    sudo apt update
sudo apt install ca-certificates curl gnupg
  2. Add Docker repo + install Docker 
    sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo \
"deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
$(. /etc/os-release && echo $VERSION_CODENAME) stable" | \
sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
  3. (Optional) Run Docker without sudo 
    sudo usermod -aG docker $USER
newgrp docker
  4. Allow access to USB devices 
    sudo usermod -aG dialout $USER
  5. Run ESPHome container without USB access 
    docker run -d \
  --name esphome \
  --restart=unless-stopped \
  -v ~/Python/ESPhome_configs:/config \
  -p 6052:6052 \
  esphome/esphome
  6. Flash new ESPs with a command line 
    docker run --rm -it \
  -v ~/Python/ESPhome_configs:/config \
  --device=/dev/ttyUSB0 \
  esphome/esphome run example.yaml
  7. Copy secrets from another computer

    Clone repo esphome to local computer. Manually create file secrets.yaml and copy the content from another computer.

  8. Open ESPHome Open ESPhome from http://localhost:6052

    Ubuntu SSH Server

  1. Install OpenSSH server 
    sudo apt update
sudo apt install openssh-server -y
  2. Enable and start SSH service 
    sudo systemctl enable ssh
sudo systemctl start ssh
sudo systemctl status ssh
  3. Edit SSH server configuration 
    sudo nano /etc/ssh/sshd_config
  4. Set secure authentication options 
    PermitRootLogin no
PasswordAuthentication no
PubkeyAuthentication yes
ChallengeResponseAuthentication no
UsePAM yes
AllowUsers ticon367
  5. Restart SSH service 
    sudo systemctl restart ssh
  6. Allow SSH through firewall 
    sudo ufw allow OpenSSH
sudo ufw enable
