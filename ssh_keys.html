SSH Key Setup

Configure key-based SSH login from your Ubuntu client (user: ticon367)

  1. Generate SSH key 
    ssh-keygen -t ed25519 -C "ticon367"
  2. Start SSH agent and add key 
    eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
  3. Copy key to server 192.168.50.140 (port 2218) 
    ssh-copy-id -i ~/.ssh/id_ed25519.pub -p 2218 ticon367@192.168.50.140
  4. Copy key to server 192.168.50.6 (port 22) 
    ssh-copy-id -i ~/.ssh/id_ed25519.pub ticon367@192.168.50.6
  5. Test SSH connection (server 140) 
    ssh -p 2218 ticon367@192.168.50.140
  6. Test SSH connection (server 6) 
    ssh ticon367@192.168.50.6
  7. Create SSH config file 
    nano ~/.ssh/config
  8. SSH config content 
    Host nas
    HostName 192.168.50.140
    User ticon367
    Port 2218
    IdentityFile ~/.ssh/id_ed25519

Host ha
    HostName 192.168.50.6
    User ticon367
    Port 22
    IdentityFile ~/.ssh/id_ed25519
  9. Use shortcuts 
    ssh nas
ssh ha

Ubuntu SSH Server

  1. Install OpenSSH server 
    sudo apt update
sudo apt install openssh-server -y
  2. Enable and start SSH service 
    sudo systemctl enable ssh
sudo systemctl start ssh
sudo systemctl status ssh
  3. Edit SSH server configuration 
    sudo nano /etc/ssh/sshd_config
  4. Set secure authentication options 
    PermitRootLogin no
PasswordAuthentication no
PubkeyAuthentication yes
ChallengeResponseAuthentication no
UsePAM yes
AllowUsers ticon367
  5. Restart SSH service 
    sudo systemctl restart ssh
  6. Allow SSH through firewall 
    sudo ufw allow OpenSSH
sudo ufw enable
