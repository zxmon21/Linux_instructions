Home Assistant VM Migration (Ubuntu 22.04 → 26.04)

  1. Shut down Home Assistant VM on old host 
    virsh shutdown homeassistant
virsh list --all
  2. Backup VM disk and configuration 
    sudo cp /var/lib/libvirt/images/homeassistant.qcow2 ~/homeassistant.qcow2
virsh dumpxml homeassistant > homeassistant.xml
  3. Export firewall rules (important for mDNS) 
    sudo iptables-save > ~/iptables-backup.rules
  4. Install virtualization stack on new host 
    sudo apt update
sudo apt install qemu-kvm libvirt-daemon-system libvirt-clients virt-manager bridge-utils
  5. Enable libvirt and add user permissions 
    sudo systemctl enable --now libvirtd
sudo usermod -aG libvirt,kvm $USER
  6. Log out and back in

    Required for libvirt group permissions to take effect.

  7. Copy VM files to new host 
    scp homeassistant.qcow2 user@NEW_PC:/var/lib/libvirt/images/
scp homeassistant.xml user@NEW_PC:~
  8. Import and start VM 
    virsh define homeassistant.xml
virsh start homeassistant
virsh list --all
  9. Ensure Ethernet is default network (not WiFi) 
    ip route
  10. Disable WiFi as default route (for inverter-only usage) 
    sudo nano /etc/netplan/01-netcfg.yaml
  11. Example Netplan config 
    network:
  version: 2
  ethernets:
    enp3s0:
      dhcp4: true

  wifis:
    wlp2s0:
      dhcp4: true
      dhcp4-overrides:
        use-routes: false
  12. Apply network config 
    sudo netplan apply
  13. Install and configure mDNS (Avahi) 
    sudo apt install avahi-daemon
sudo nano /etc/avahi/avahi-daemon.conf
  14. Allow both interfaces for mDNS 
    [server]
allow-interfaces=enp3s0,wlp2s0
use-ipv4=yes
use-ipv6=no
  15. Restart Avahi 
    sudo systemctl restart avahi-daemon
  16. Allow mDNS in firewall (critical) 
    sudo iptables -A INPUT -p udp --dport 5353 -d 224.0.0.251 -j ACCEPT
sudo iptables -A OUTPUT -p udp --sport 5353 -d 224.0.0.251 -j ACCEPT
  17. Persist firewall rules 
    sudo apt install iptables-persistent
sudo netfilter-persistent save
  18. Test Home Assistant 
    ping <HA-IP>
  19. Open Home Assistant Open Home Assistant
  20. Test mDNS discovery 
    ping homeassistant.local
avahi-browse -a
  21. Test inverter connectivity via WiFi 
    ping <INVERTER-IP>
