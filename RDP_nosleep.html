Ubuntu 26.04: Prevent Sleep + Lid Ignore + RDP Server Setup

  1. Install RDP server (xrdp) 
    sudo apt update
sudo apt install xrdp xorgxrdp -y
  2. Enable and start RDP service 
    sudo systemctl enable xrdp
sudo systemctl start xrdp
  3. Allow RDP through firewall (if UFW is active) 
    sudo ufw allow 3389/tcp
  4. Verify RDP service is running 
    systemctl status xrdp
  5. Find your local IP address 
    ip a

    Look for something like 192.168.x.x

  6. Prevent lid close from affecting power 
    sudo nano /etc/systemd/logind.conf

    Add or modify:

    HandleLidSwitch=ignore
HandleLidSwitchExternalPower=ignore
HandleLidSwitchDocked=ignore
  7. Restart logind to apply lid settings 
    sudo systemctl restart systemd-logind
  8. Disable suspend, hibernate and sleep states 
    sudo systemctl mask sleep.target suspend.target hibernate.target hybrid-sleep.target
  9. Disable GNOME idle sleep 
    gsettings set org.gnome.settings-daemon.plugins.power sleep-inactive-ac-type 'nothing'
gsettings set org.gnome.settings-daemon.plugins.power sleep-inactive-battery-type 'nothing'
gsettings set org.gnome.desktop.session idle-delay 0
  10. Disable screen lock (optional but recommended for RDP use) 
    gsettings set org.gnome.desktop.screensaver lock-enabled false
  11. Connect via RDP from another machine 
    :3389

    Use Windows Remote Desktop, Remmina, or Microsoft Remote Desktop.

  12. Check everything is working 
    systemctl status xrdp
ip a
systemctl status systemd-logind
  13. Notes

    - xrdp works best with Xorg sessions (default on Ubuntu 26.04)
    - Masking sleep targets fully prevents suspend even on laptop lid close
    - If you want Wayland RDP later, different setup is required
