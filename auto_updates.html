Automatic Updates (Ubuntu)

Configure automatic security updates, optional full upgrades, and Docker image updates.

1. Install unattended upgrades 
sudo apt update
sudo apt install unattended-upgrades apt-listchanges
2. Enable automatic security updates 
sudo dpkg-reconfigure unattended-upgrades
3. Ensure automatic upgrades are active 
sudo nano /etc/apt/apt.conf.d/20auto-upgrades
4. Set config content 
APT::Periodic::Update-Package-Lists "1";
APT::Periodic::Unattended-Upgrade "1";

Optional: Full system upgrades (weekly)

5. Create systemd timer for full upgrades 
sudo nano /etc/systemd/system/full-upgrade.service
6. Service content 
[Unit]
Description=Full system upgrade

[Service]
Type=oneshot
ExecStart=/usr/bin/apt update
ExecStart=/usr/bin/apt -y full-upgrade
ExecStart=/usr/bin/apt -y autoremove
7. Create timer 
sudo nano /etc/systemd/system/full-upgrade.timer
8. Timer schedule (weekly) 
[Unit]
Description=Run full system upgrade weekly

[Timer]
OnCalendar=weekly
Persistent=true

[Install]
WantedBy=timers.target
9. Enable timer 
sudo systemctl daemon-reload
sudo systemctl enable --now full-upgrade.timer

Optional: Auto-update Docker containers (ESPHome etc.)

10. Install Watchtower 
docker run -d \
  --name watchtower \
  --restart=unless-stopped \
  -v /var/run/docker.sock:/var/run/docker.sock \
  containrrr/watchtower \
  --cleanup --interval 86400
